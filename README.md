# NewsPaper

## 💻프로젝트 소개 
1. 전자신문 섹션 중 산업에너지 환경에서 과학, 에너지 분야 크롤링하여 csv 파일을 작성한다.
2. 수집한 csv 파일에서 에너지 관련 단어의 빈도수를 도출한다
3. 에너지 관련 단어에 대하여 적절한 통계량(평균, 표준편차, 최빈단어)을 구하기
4. 주어진 데이터를 활용하여 최대한 다양한 방법으로 데이터를 표현해 보기

📌 시행 목적 
- 데이터 수집과 분석을 익히기 위해 실습 하고자 함

⚙ 개발환경
-  python
